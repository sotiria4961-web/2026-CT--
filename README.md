# 🎮 A+를 향해 달려라!

Pygame으로 제작된 **횡스크롤 러닝 액션 게임**입니다.
플레이어는 장애물을 피하고 플랫폼을 넘으며 달려 나가 **챕터를 진행**하고, 점점 빨라지는 속도 속에서 최대한 오래 살아남는 것이 목표입니다.

---

## 📌 게임 개요

* **장르**: 2D 횡스크롤 러닝 / 플랫폼 액션
* **플랫폼**: PC (Python + Pygame)
* **게임 방식**: 자동 전진형 러너
* **목표**: 장애물을 피하며 챕터를 진행하고 최고 기록 달성

---

## 🕹️ 조작 방법

| 키     | 동작            |
| ----- | ------------- |
| ← / → | 좌우 이동         |
| 스페이스바 | 점프            |
| ↓     | 낙하 가속 / 빠른 착지 |
| ESC   | 게임 종료         |

> ※ 일부 장애물은 **특정 점프 타이밍**을 요구합니다.

---

## 🧍 캐릭터 시스템

* 여러 캐릭터가 존재하며, 각 캐릭터는 **색상으로 구분**됩니다.
* 캐릭터 선택은 게임 시작 시 자동 혹은 랜덤으로 이루어집니다.

---

## 🧱 플랫폼 & 장애물

### 플랫폼

* 일반 플랫폼
* 길이와 높이가 랜덤으로 생성

### 장애물 종류

* **force_jump**: 반드시 점프를 강요하는 장애물
* **tall_jump**: 높은 점프가 필요한 장애물

> 게임이 진행될수록 **장애물 생성 빈도와 속도가 증가**합니다.

---

## 📖 챕터 시스템

* 게임은 여러 **챕터(Chapter)** 로 구성되어 있습니다.
* 챕터가 바뀔수록:

  * 이동 속도 증가
  * 장애물 생성 간격 감소
  * 난이도 상승

챕터 전환 시 **로딩/전환 애니메이션**이 재생됩니다.

---

## 🎵 사운드 & 효과

* 점프, 충돌, 게임 오버 등 상황별 효과음 포함
* 배경음악(BGM) 지원

> 사운드 파일은 `assets/` 폴더에 위치해야 합니다.

## ⚙️ 실행 방법

### 1️⃣ 필수 환경

* Python 3.9 이상
* pygame 라이브러리

```bash
pip install pygame
```

```bash

```
